# 朝の忘れ物チェック

HTML・CSS・バニラ JavaScript で作られた、朝の持ち物確認に使えるチェックアプリです。
インストール不要で、`index.html` をブラウザで開くだけですぐ使えます。

🔗 **デモ**: [こちらから確認できます](https://vaishnav-sh.github.io/todo-app/)

## 機能一覧

| 機能 | 説明 |
|------|------|
| 項目の追加 | 入力欄から自由に追加 |
| クイック追加 | よく使う持ち物をワンタップで追加 |
| 重要表示 | 優先して確認したい項目を目立たせる |
| 項目の編集 | 項目名をその場で変更 |
| 確認済み / 戻す | 確認済みにしたり未確認へ戻したりできる |
| 項目の削除 | 不要な項目を個別削除 |
| 一括リセット | すべてのチェック項目をまとめて削除 |
| 進捗サマリー | 全体数・未確認数・完了率を表示 |
| 完了一覧 | 確認済み項目を折りたたんで管理 |
| 達成メッセージ | すべて確認し終えると完了メッセージを表示 |
| テーマ切替 | 明るい表示 / 夜向け表示を切り替え |
| 自動保存 | `localStorage` に保存され、再読み込み後も維持される |

## ローカルで動かす

ビルドツール・パッケージマネージャー不要です。

**1. リポジトリをフォークしてクローン**

SSH の場合:
```bash
git clone git@github.com:<your_github_username>/todo-app.git
```

HTTPS の場合:
```bash
git clone https://github.com/<your_github_username>/todo-app.git
```

**2. `index.html` をブラウザで開く**

```bash
cd todo-app
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

## 使い方マニュアル

### 項目を追加する

1. 画面上部の入力欄に確認したい持ち物や準備項目を入力する
2. **「追加する」ボタン**（または `Enter` キー）を押すと一覧に追加される
3. 空白のみの入力は無視される

---

### よく使う持ち物をすぐ追加する

- 入力欄の下にある **候補ボタン** を押すと、`鍵` や `財布` などをすぐに追加できる
- 毎朝よく確認する項目を素早くリスト化したいときに便利

---

### 項目を確認済みにする

- 項目右側の **「確認済み」ボタン** を押すと、その項目が確認済み一覧へ移動する
- 進捗サマリーの **全体数・未確認数・完了率** がリアルタイムで更新される
- **すべての項目を確認**すると、画面下部に完了メッセージが表示される

---

### 優先して確認したい項目を目立たせる

- 項目右側の **「☆ 重要」ボタン** を押すと、その項目が優先表示になる
- 優先項目は一覧の上に寄り、見逃しにくくなる
- もう一度押すと解除できる

---

### 項目名を編集する

1. 項目右側の **「編集」ボタン** を押す
2. 入力欄に切り替わるので内容を書き換える
3. **「保存」ボタン** または `Enter` キーで保存する
4. `Esc` キーまたは **「キャンセル」ボタン** で編集をやめられる

---

### 確認済み項目を未確認に戻す

- 確認済み一覧にある **「未確認に戻す」ボタン** を押すと、再び未確認の一覧へ戻せる
- 押し間違えたときのやり直しに使える

---

### 項目を削除する

- 未確認・確認済みのどちらの一覧でも **「削除」ボタン** から個別に削除できる

---

### すべての項目をまとめて削除する

- 一覧右上の **「すべてリセット」ボタン** を押すと、全項目をまとめて削除できる
- 確認メッセージが出るので、誤操作もしにくい

---

### 確認済み一覧を開閉する

- **「確認済み」セクション** を押すと、完了した項目の一覧を開閉できる
- 開閉状態はブラウザに保存されるため、次回も使いやすい

---

### テーマを切り替える

- 右上の **テーマ切替ボタン** で、明るい表示と夜向け表示を切り替えられる
- 設定はブラウザに保存されるため、次回もそのまま表示される

---

### データの保存について

- 項目の追加・確認済み・戻す・削除・編集を行うたびにブラウザの `localStorage` に自動保存される
- ページを再読み込みしても項目は消えない
- データはブラウザ・デバイスごとに保存されるため、別のブラウザやデバイスには同期されない
